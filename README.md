# Soraq
Validate Information Architecture before building products.
Soraq is a UX Research platform focused on Card Sorting and Tree Testing for modern digital products.
Designed for UX researchers, product designers and teams that want to understand how users organize, navigate and interpret information structures.

## Features
* Card Sorting studies
* Tree Testing
* Automatic clustering
* Similarity matrices
* Dendrogram analysis
* Shareable participant links
* No participant registration required
* LATAM-first experience

## Why Soraq?
Most UX research tools are:

* expensive,
* enterprise-oriented,
* English-first,
* overloaded with unnecessary complexity.

Soraq focuses on simplicity, speed and actionable insights.

## Philosophy
Soraq does not sell "Card Sorting".

It sells:
* clarity,
* validation,
* faster UX decisions,
* reduced redesign costs,
* evidence over assumptions.
